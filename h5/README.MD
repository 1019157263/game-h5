# FLY飞机大战简介
-------
## 1.快速开始
### 1.1用浏览器打开1.html来运行本程序
	对1.html点击右键，选择用浏览器打开即可

-------
## 2.文件介绍
### 2.1、1.html
	1.html为主界面，为程序入口和容器

### 2.2、1.js
	2.js为程序的逻辑和小游戏的实现全靠1.js

-------
## 3.对象和方法介绍
### 3.1、yuan对象
	yuan对象为本游戏玩家控制的对象，包含3个属性x、y、r分别是坐标和半径
### 3.2、sw对象
	sw对象为玩家需要消灭的敌人对象
